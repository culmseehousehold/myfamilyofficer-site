# Australian Investment Manager (FamilyOfficer)

## The Operating System for Australian Wealth

Consolidate your entire wealth ecosystem—trusts, companies, and portfolios—into a single, real-time source of truth.

### Features

- **Multi-Entity Wealth Structures**: Visualize and manage complex webs of Discretionary Trusts, Unit Trusts, Companies, and SMSFs.
- **Institutional-Grade Portfolio Management**: TWR, IWRR/XIRR, and multi-currency support.
- **Automated Compliance & Tax**: optimize tax outcomes and automate SMSF compliance.
- **AI-Powered "Zero-Touch" Admin**: Drag-and-drop PDF extraction and email ingestion.
